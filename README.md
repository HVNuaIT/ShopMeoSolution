# ASP.NET CORE 6.0 project web bán mèo của  HVNuaIT
## Công nghệ sữ dụng 
- ASP.NET CORE 6.0
- Entity Framework Core 6.0
- SQLSERVER
- Linq
- JWT 
## Một số chức năng 
- CRUD Sản Phẩm,Danh Mục,Order Sản Phẩm...
- Tích Hợp Plugin Chat Fb.
- Xem chi tiết.
- ...
## Hướng phát triển Project 
- Tích hợp chức năng thanh toán online cho khách.
- Deloy lên host (Chỉ chạy trên IIS).
